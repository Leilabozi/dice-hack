## Usage
Launch the app. <br/>
Enter your custom labels/tasks for each dice side.<br/>
Press Next to go to the dice roll screen.<br/>
Tap Roll! to animate and randomly select a side.<br/>
View the result and tap Roll Again to try another.<br/>
